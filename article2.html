ความหมายของดอกไม้แต่ละประเภท1/2

ข้อมูลเกี่ยวกับความหมายของดอกไม้แต่ละประเภท

Rose ( กุหลาบ )

เป็น ราชินีแห่งบรรดาดอกไม้ และ ตัวแทนของความรักหลายๆ แบบ ไม่ว่าจะเป็นคนรัก คนสำคัญ ครอบครัว เพื่อนสหาย

แต่คงไม่พ้นเรื่องของความรัก เพราะกุหลาบถือเป็นสัญลักษณ์ของความรักและความโรแมนติก รวมไปถึงตำนานของดอกกุหลาบ บางตำนานเชื่อว่า กุหลาบเสมือนเครื่องหมายแทนการกำเนิดของ เทพธิดาวีนัส เทพแห่งความงามและความรัก

Cutter flower ( คัดเตอร์ )

กำเนิดมาจากอเมริกาเหนือ ซึ่งมีหากหลายสีไม่ว่าจะเป็นสีขาว สีเหลือง หรือสีม่วง โดยดอกคัตเตอร์สีขาวหรือ White Cutter Flower ถึงแม้จะไม่มีกลิ่นหอมที่แรงมาก แต่กลิ่นที่ออกมานั้นเป็นกลิ่นธรรมชาติ

ยังถือว่าเป็นดอกไม้ที่เสียสละ แต่ความหมายของดอกคัตเตอร์นั่นกลับลึกซึ้ง เพราะแปลว่า “แม้คุณจะไม่มองฉันก็ไม่เป็นไร แต่ฉันก็จะมีแต่เพียงคุณเท่านั้น” ซึ่งเหมาะแก่การแทนใจสื่อให้แก่คนที่แอบชอบ

Lily ( ลิลลี่ )

ดอกลิลลี่ เป็นดอกไม้ที่ใครๆ มักจะรู้จักและคุ้นเคย ซึ่งมีแต่ละสีมีความหมายแตกต่างกัน

ดอกสีขาว แปลว่า ความบริสุทธิ์ใจ ไร้เดียงสา อ่อนน้อมและถ่อมตน

ดอกสีชมพู คือ ตัวแทนของความรัก เสน่หา ความเข้าอกเข้าใจ ความเป็นผู้หญิง และยังสื่อได้ถึงหญิงสาววัยแรกแย้ม ที่เต็มไปด้วยเสน่หาอันน่าหลงใหล

ดอกสีเหลือง ตัวแทนของความสนุกสนาน ความรื่นเริงบันเทิงใจ มิตรภาพ แปลได้ว่า “ฉันดีใจที่มีคุณในชีวิต ฉันอยากให้คุณยิ้ม ฉันอยากให้คุณมีความสุข เพราะฉันรักคุณมากจริง ๆ”

ดอกสีส้ม เป็นตัวแทนถึงความร่าเริง สดใส ความปิติสุขที่ได้อยู่ใกล้ แปลได้ว่าคือ “สุขใจที่ได้ใกล้เธอ”

ดอกสีม่วง ตัวแทนของ ความเคารพ ความสำเร็จ และความพิเศษที่ไม่เหมือนใคร สามารถสื่อความหมายได้ว่า “คุณคือคนพิเศษของฉัน ฉันอยากหยิบยื่นแต่สิ่งดี ๆ ให้กับคุณ”

Carnation ( คาร์เนชั่น )

ดอกคาร์เนชั่น เป็นสัญลักษณ์ของการเฉลิมฉลอง ซึ่งในตำนานความเป็นมาของชาวกรีกโรมันมักจะนิยมในการใช้เจ้าดอกไม้ชนิดนี้ในการแสดงความยินดี ความรื่นเริงต่างๆ
