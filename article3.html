ความหมายของดอกไม้แต่ละประเภท1/2

ข้อมูลเกี่ยวกับความหมายของดอกไม้แต่ละประเภท

Forget me not ( ฟอร์เก็ตมีน็อต )

ดอกไม้ที่มีความหมายแสนโรแมนติก เป็นดอกไม้ขนาดเล็กและเป็นที่นิยมกับคู่รักที่ใช้มอบให้กัน

โดยตำนานล่าว่า ที่มาของชื่อดอกไม้เกิดจากมีอัศวินหนุ่มในชุดเกราะกำลังเดินเล่นที่ริมธารกับหญิงสาวอันเป็นที่รักของเขา เมื่อสาวเจ้าเห็นดอกไม้ที่ริมธารชูช่อออกดอกสวยงาม ก็เกิดอยากจะได้เจ้าดอกไม้นั้นขึ้นมา ดังนั้นอัศวินหนุ่มจึงอาสาเก็บให้ แต่กลับลื่นตกลงไปในธารน้ำ ซึ่งชุดเกราะหนัก ทำให้เขาไม่สามารถว่ายน้ำได้ เขาจึงตัดสินใจโยนดอกไม้นั้นให้เธอ พร้อมกับตะโกนว่า Forget Me Not (อย่าลืมฉัน) และจมหายไป นั่นจึงเป็นตำนานที่ทุกคนต่างเล่าขานถึงความโรแมนติกดอกฟอร์เก็ตมีน็อต

Daisy ( เดซี่ )

ดอกไม้ขนาดเล็ก ที่มักถูกซ่อนอยู่ในงานแต่ง ลักษณะเป็นกลีบน้อยๆ และช่วยสร้างสีสันให้กับดอกไม้อื่นๆ หลายคนคงคุ้นตากับดอกเดซี่เพราะงานแต่ง แต่ในควาจริงแล้ว ดอกเดซี่เป็นสัญลักษณ์แทนหัวใจอันบริสุทธิ์ และดอกน้อยๆ ยังนิยมใช้เพื่อทำนาย รัก-ไม่รัก

Tulip ( ทิวลิป )

ดอกทิวลิป มีความหมายถึง การตกหลุมรักอย่างหัวปัักหัวปำ เป็นรักแรก บ่งบอกได้ว่า “ฉันตกหลุมรักเธออย่างหมดหัวใจ” และ เป็นสัญลักษณ์ของจินตนาการ ความใฝ่ฝัน คู่รักที่สมบูรณ์แบบและรักที่เปิดเผย

Gerbera ( เยอบีร่า )

ดอกไม้ที่เต็มไปด้วยสีสันสดใจร่าเริง ความหมายของดอกเยอบีร่านั้นคือ ความบริสุทธิ์ไร้เดียงสา และความเข้มแข็ง จึงแปลได้ว่า “จิตใจที่เต็มไปด้วยความบริสุทธิ์ไร้เดียงสา ที่แฝงไปด้วยความเข้มแข็ง” หรือ “เธอคือแสงอาทิตย์แห่งชีวิตของฉัน”

Gypso ( ยิปโซ )

ดอกยิปโซ แม้จะไม่เป็นดอกไม้ที่โดดเด่นนัก หรือมักถูกนำมาใช้เป็นดอกประดับเช่นเดียวกับดอกคัตเตอร์ แต่ความหมายของดอกยิแโซคือ ความบริสุทธิ์ รักแรกพบ จึงนิยมถูกนำไปมอบให้แก่คนที่เรากำลังตกหลุมรักตั้งแต่ครั้งแรก

Sunflower ( ทานตะวัน )

ดอกทานตะวัน หมายถึง “เธอคือความสดใสในชีวิตฉัน”

ดอกทานตะวัน ที่มีมีใบล้อมรอบจานดอก มีสีเหลืองสดใส จนถูกเปรียบเป็นดั่งพระอาทิตย์ แม้ว่าดอกทานตะวันจะไม่ได้มีสีอื่นๆ เหมือนดอกไม้ดอกอื่น แต่จำนวนของดอกทานตะวันต่างหาก ที่มีความหมาย

ดอกทานตะวัน หมายถึง รักของฉันมั่นคงและภักดีต่อเธอเสมอ ดุจดั่งทานตะวันที่ไม่เคยหันมองผู้ใดนอกจากดวงอาทิตย์

Hydrangea ( ไฮเดรนเยีย )

ดอกไฮเดรนเยีย มีความหมายทั้งในเชิงลบ และเชิงบวก แต่ความหมายที่แท้จริงของดอกไฮเดรนเยีย หมายถึง ‘หัวใจที่ด้านชา’ และเพราะแบบนั้น ในอีกนัยคือ คือแทนคำขอบคุณ จากผู้ที่มีหัวใจด้านชา กับประโยคสำคัญที่ว่า “ขอบคุณที่เข้าใจกัน”
